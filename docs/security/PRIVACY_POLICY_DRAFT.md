# Privacy Policy Draft

This is a draft for the future production service. Review legally before public launch.

## Data Collected

- Telegram user id and username.
- URLs submitted for download.
- Download job status and error codes.
- Optional encrypted platform sessions if the user connects an account.

## Data Not Collected Intentionally

- Plaintext passwords.
- Plaintext cookies.
- Private messages outside interactions with the bot.

## Session Use

Connected sessions are used only to access content that the user's own platform account is authorized to view.

## Data Retention

- Temporary media files are deleted after delivery or after a configured TTL.
- Download history may be retained for debugging and user history.
- Users can revoke connected sessions.

## User Controls

Users can disconnect sessions using `/disconnect`.

## Abuse

The service may rate-limit or block usage that appears abusive, illegal, or privacy-invasive.
